## 一、为什么做 Run 3？

### Run 2 的核心问题

Run 2（GPT-5.4 judge + lr=5e-7）的 avg_loss 曲线一直下降，**没有出现 U 形回升**，模型处于"被推走但没学会"的状态。在未收敛时评测，结果不能反映方法真实上限。

### 诊断：为什么 Run 2 没有 U 形？

```
U 形出现的条件：
  梯度更新速度（lr）> teacher-student gap 扩大速度

Run 1（self-judge, lr=5e-7）：gap 小 → lr 够 → step ~30 出现 U
Run 2（GPT judge,  lr=5e-7）：gap 大 → lr 不够 → 110 步无 U

GPT-5.4 产生的 gap 约是 self-judge 的 3-4 倍
→ 需要 lr 至少翻倍（5e-7 × 2 = 1e-6）才能在 110 步内看到 U 形
```

## 四、训练曲线深度分析

### avg_loss 走势

```
step  1：-0.660（起点）
step ~20：-0.760（第一次最低）← lr 更大，下降更陡、更快
step ~40：-0.710（第一次回升高点）← U 形右侧出现 ✅
step ~80：-0.760（第二次最低）
step 116：-0.752（终点，微弱回升）
```

**形态：W 形（双 U）**

### 三轮 avg_loss 对比

| | Run 1（self-judge, lr=5e-7）| Run 2（GPT, lr=5e-7）| Run 3（GPT, lr=1e-6）|
|--|--|--|--|
| 起点 | -0.760 | -0.690 | -0.660 |
| 最低 | -0.860（step ~30）| -0.750（step ~30）| **-0.760（step ~20）** |
| 终点 | -0.840（回升 0.02）| -0.750（无回升）| **-0.752（回升 0.008）** |
| U 形 | 浅 U | 无 | **W 形（双 U）** |
| 评价 | 初步收敛 | 未收敛 | **部分收敛** |

### 为什么是 W 形而不是单 U？

W 形（双 U）是多阶段收敛的正常表现：

```
第一个 U（step 1~40）：
  前半段训练数据（约前 1000 条）触发 → 推开 → 追上

第二个 U（step 40~116）：
  后半段训练数据（约后 1000 条，不同科室分布）→ 再次推开 → 开始追
  但 116 步结束，第二个 U 未走完
```

**W 形 ≠ 训练失败**，而是"两阶段学习"的体现，本质上是 110 步内数据分布变化导致的多次收敛迭代。

### lr=1e-6 vs lr=5e-7 的量化对比

| | lr=5e-7（Run 2）| lr=1e-6（Run 3）|
|--|--|--|
| 第一个 U 出现 | 无 | **step ~20**（提前）|
| 第一个 U 回升幅度 | 0 | **+0.050** |
| 说明 | 完全未收敛 | **部分收敛** |

**结论**：lr 翻倍有效，方向正确。但要在 110 步内看到单 U 完整收敛，需要 lr=2e-6（见 Run 4 计划）。

### 结果解读

**1. Run 3 总体最佳（-0.031）**
三轮中下降幅度最小，且 5 个科室有正向提升（Run 1/2 各 3 个）。

**2. 持续改善科室**
神经、内分泌代谢、检验影像、急诊重症、感染免疫在三轮中单调改善，说明 lr 提升方向始终有效。

**3. 药理用药异常（Run3 -0.06 vs Run2 +0.15）**
Run 2 药理效果最好，Run 3 反而退步。原因：lr 更大导致第二个 U 的推偏阶段影响了药理知识的稳定性，未等模型重新追上就结束了。这是 W 形未完成时的副作用。

**4. 整体趋势**
```
随 lr 增大：-0.053 → -0.070 → -0.031
            Run1    Run2（未收敛）  Run3（部分收敛）
```
Run2 比 Run1 更差是因为未收敛，而非方法本身更差。

## 八、经验总结

1. **lr 是 OPSD 收敛的核心杠杆**：judge 质量越高 → gap 越大 → 需要更高 lr。GPT judge 需要 2 倍以上的 lr 才能在同等步数内开始收敛。

2. **W 形是多阶段收敛的正常现象**：不同批次数据难度不同会产生多次"推开-追上"循环，不等于训练失败。

3. **未收敛时不应做最终评测**：Run 2（未收敛）评测结果比 Run 1（部分收敛）更差，但这是因为评测时机错误，不是方法本身更差。

4. **单变量实验的价值**：Run 2→Run 3 只改 lr，使得结论干净：lr 提升确实改善了收敛和效果。

5. **接受率不等于训练信号质量**：Run 3 接受率 92.8%（高于 Run 1/2 的 88%），但不代表每个 sample 的 hint 都高质量——GPT-5.4 可能对简单问题过于宽松地给正分。
